# Flujo TV Premium Latinoamérica

Landing oficial de Flujo TV Premium para LATAM, construida con Astro + Tailwind.

## Dominio oficial

- https://premium.flujooficial.com

## Stack

- Astro 5
- Tailwind CSS
- TypeScript

## Características principales

- Landing orientada a conversión (CTA a WhatsApp)
- SEO técnico (canonical, hreflang, OG/Twitter, schema)
- Sitemap y robots configurados para indexación
- Diseño responsive para móvil, tablet y desktop

## Compatibilidad de dispositivos

- Android (móvil y tablet)
- TV Box
- Fire Stick
- Google TV
- Xiaomi Stick

## No compatible de forma nativa

- iOS / iPhone / iPad
- Apple TV
- Roku
- Smart TV Samsung/LG sin TV Box

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

## Archivos SEO clave

- `astro.config.mjs`
- `src/config/site.ts`
- `public/robots.txt`
- `public/sitemap.xml`
- `src/layouts/Layout.astro`

## Configuración de analítica (opcional)

Crear `.env` tomando como base `.env.example`:

- `PUBLIC_GA_MEASUREMENT_ID`
- `PUBLIC_META_PIXEL_ID`
